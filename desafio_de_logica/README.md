# Desafio Lógica - Seletiva Olimpíeda

## Desafio 1 - Fila do recreio

|Nome|CPF|
|-|-|
|Maria|123|
|Juliana|4564|
|Ana|454654|
|Bruna|25|

|Bruna|25|

|Maria|123|
